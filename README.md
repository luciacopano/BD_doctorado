# BD_doctorado
Base de datos realizada en prácticas en la Universidad de Cádiz.

Este proyecto consiste en la realización de una BDD en phpmyadmin para la asignatura 'Bases de Datos' del Grado en Ingeniería Informática en la Universidad de Cádiz.
Contiene los códigos sql para la creación de las tablas, un gist para poder usar álgebra relacional en ella y documentación sobre su creación.
